# 数学计算

在一条记录中我们可以对字段进行数学运算，如下：

```
SELECT num1+num2 AS sum
FROM demo2;
```

返回了sum列，内容是num1+num2的结果。但是查看数据库，可以发现数据库内的列并没有被写入数据，说明这样的运算是一次性的。
