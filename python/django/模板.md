模板就是静态模板，存放的是静态资源。

# 动态静态交互

交互的主要方法就是做静态渲染，做静态渲染的时候传送一个字典给静态模板，模板对数据进行解析。

```python
return render(request, 'home.html', {'string': string})
```

这就是返回了一个字典给了静态模板。模板要输出这个字典，使用```{{string}}```输出这个字典。

# 静态操作

## for循环

```html
        {% for foo in test %}
            {{ foo }}
            <br>
        {% endfor %}
```

这循环输出了传入的列表。
